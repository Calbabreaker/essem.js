Public Properties

active 
active

Whether or not the entity is active. Making the entity unactive will remove it from systems and the scene tag collection and put back when active. Setting the value will make all its children be the same active state unless the child is explicitly set to be not active and the parent(s) is set to be active.

activeSelf 
activeSelf

The local active state. This will be regardless of it's parents.
